Шаблоны референсов для режима «Фото по референсу».

Как обновлять картинки:
1. Положите сюда файлы изображений (jpg, png и т.д.).
2. Укажите имена файлов в public/ref-templates.json в поле "image", например:
   "ref-templates/ваш-файл.jpg"
3. Добавить новый шаблон: добавьте в JSON объект { "name": "Название", "image": "ref-templates/файл.jpg" }.
4. Удалить шаблон: удалите строку из ref-templates.json.
5. Обновите страницу в приложении — список подтянется заново.
